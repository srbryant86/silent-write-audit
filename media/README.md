# Walkthrough video

`silent-write-audit-walkthrough.mp4` — a 90-second AI-narrated walkthrough of the bug class, what we found in our own codebase, what the audit-as-a-service delivers, and pricing.

## How it was built

The MP4 is generated by `build/build-walkthrough.py`. Pipeline:

1. Per-scene narration audio via [edge-tts](https://github.com/rany2/edge-tts) (Microsoft Edge TTS, free, no API key) using the `en-US-AndrewNeural` voice.
2. Per-scene slide PNGs rendered via PIL using a slate-900 / emerald-500 palette matching the README aesthetic.
3. Each scene composited as an MP4 (still image + audio) via ffmpeg.
4. Scenes concatenated into the final MP4 via ffmpeg.

## Re-generating the video

```bash
cd media/build
pip install edge-tts pillow
python build-walkthrough.py
```

ffmpeg must be on PATH. Output lands at `../silent-write-audit-walkthrough.mp4`.

## Editing the script

The narration text and slide layouts live inline in `build-walkthrough.py` (one function per scene). To change a scene:

1. Edit the `NARRATION_N` string and/or `scene_N_*()` function.
2. Re-run `python build-walkthrough.py`.
3. Commit the regenerated MP4.

The intermediate PNG / MP3 / per-scene MP4 files are gitignored — only the source script and final concatenated MP4 ship in the repo.

## Voice options

Default is `en-US-AndrewNeural` (warm, confident, technical-friendly). Other reasonable choices:

- `en-US-GuyNeural` — slightly more formal
- `en-US-DavisNeural` — more conversational
- `en-US-BrianNeural` — energetic

Change the `VOICE` constant at the top of the script.

## Why AI-voiced rather than human-recorded

The 2026-05-06 venture-optimizer audit's chosen recommendation called for a 90-second human-recorded Loom. Operator opted for AI-voiced as a lower-friction equivalent; quality is acceptable for technical-audience developer-tool walkthroughs, and the time-cost was zero ongoing (re-record any time the script changes by re-running the build). The brand promise about "human triage" applies to the audit work itself, not the marketing copy — AI-voiced product walkthrough about a human-delivered service is fine.
